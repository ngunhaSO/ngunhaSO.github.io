TEAM PROJECTS
Annual Forms Submittal System

• A web application for 39 counties to submit various reports four times a year. CRAB management staff will review and approve the county reports through this system.

• Technologies used: ASP.Net MVC/WebApi, LINQ, SQL Server, AngularJS, HTML5, Bootstrap, CSS3

Mobility

• An inventory management system for the county road infrastructure and county road related inventories, e.g., road intersections, bridges, guardrails, etc.

For now, the Mobility has 25 managed inventories.

• Technologies used: VB.Net, ADO.Net data service, LINQ, SQL Server

Systemic Safety Project Selection Tool

• Provides a step by step process for conducting systemic safety analysis; analytical techniques for determining a reasonable balance between the implementation of spot safety improvements and systemic safety improvements, and a mechanism for quantifying the benefits of safety improvements implemented through a systemic approach.

• Technologies used: VB.Net, ADO.Net data service, LINQ, SQL Server

County Transportation Performance Metrics

• A graphical web-based dashboard of county transportation performance measures. A data warehouse was created for storing metric data from multiple sources, both from within our agency, and from outside agencies. Data import procedures where documented and a schedule was created to keep the data store as up to date as practical.

• Technologies used: ColdFusion, SQL Server, jQuery, CSS3

Query Builder

• This is a light weight query builder that allows non programmers to query a database easily for detailed analysis. The end user only needs to select columns from tables/views/table functions in a tree view to build the SQL Server query statement. The tool also supports column name aliasing, grouping, sorting and filtering. The user can also save current selection to the Desktop in a JSON format for later use.

• Technologies used: C#, SQL Server, LINQ
